Esta es una aplicación desarrollada para un proyecto de tesis universitaria para la Universidad Alejandro de Humboldth
en Caracas, Venezuela. Este proyecto inicialmente tiene como proposito conectarse a un dispositivo Arduino y así
poder controlar las luces de una casa. Sim embargo, al terminar el código espero poder separarlo para que pueda ser 
usado como una interfaz de comunicación fácilmente acoplable a cualquier otra aplicación de Android que desee usar el bluetooth.
